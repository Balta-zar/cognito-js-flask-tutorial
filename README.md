#### Install dependencies
Python: <br/>
pip install -r requirements.txt
<br/><br/>
Node.js: <br/>
cd pem/ <br/>
npm install
#### Start project
python app.py <br/><br/>
Don't forget to fill your <br/> 
UserPoolId (static/app.js), <br/>
ClientId (static/app.js), <br/>
pem (app.py) and <br/>
url (pem/index.js)
